Control Layout plan:

Driver Xbox:

LeftY: translate up/left
LeftX: translate left/right
RightX: rotate Z

??? Unlock Climber solenoids
L/R triggers: L/R climber UP
L/R bumpers: L/R climbers DOWN

Operator Xbox:
X: intake
A: amp score position
B: close score position
Y: far score position

L bumper: spin up shooter
L trigger: spin up shooter

LeftY: run intake manually
???: run shooter backwards


Auto Combinations

SingleStation -> 2 Note
SingleStation -> 3 Note
Middle-> 2 Note
Middle-> 3 Note SingleSide
Middle-> 3 Note DoubleSide
DoubleStation -> 2 Note
DoubleStation -> 3 Note
